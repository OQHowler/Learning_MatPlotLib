# 📊 Learning Matplotlib (Visualization Practice Repository)

## 🚀 Overview

This repository contains my **hands-on, code-first approach to learning
Matplotlib**, focusing on building intuition for data visualization.


------------------------------------------------------------------------

## 📂 Repository Structure

The learning progression is as follows:

1.  **Plots.py**\
    → Basic line plots

2.  **BarChart.py**\
    → Bar charts and categorical visualization

3.  **PieChart.py**\
    → Pie charts and proportions

4.  **ScatterGraph.py**\
    → Scatter plots and relationships

5.  **Histogram.py**\
    → Distribution visualization

6.  **SubPlots.py**\
    → Multiple plots in one figure

7.  **Pandas_plt.py**\
    → Using Matplotlib with Pandas

8.  **poke_data.csv**\
    → Dataset used for visualization

------------------------------------------------------------------------

## 🧠 Learning Approach

-   Learn by **building visualizations directly**
-   Each file isolates **one plotting concept**
-   Progression from **basic → structured visualization**
-   Focus on understanding **when and why to use each plot**

------------------------------------------------------------------------

## ⚙️ How to Use

Run scripts individually:

``` bash
python Plots.py
```

Follow the order listed above for best understanding.

------------------------------------------------------------------------

## 🔥 Key Concepts Covered

-   Line plots and trends\
